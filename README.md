# PS5 Gaming Hub 🎮

Un site web vitrine moderne et fluide dédié à l'univers de la PlayStation 5. Ce projet utilise du HTML5, CSS3 et JavaScript natif.

## Fonctionnalités
- Design responsive (s'adapte aux mobiles et ordinateurs).
- Thème sombre inspiré de l'interface PlayStation.
- Système interactif de "Likes" en JavaScript sur les jeux.

## Comment lancer le projet localement
1. Téléchargez ou clonez ce dépôt.
2. Ouvrez le fichier `index.html` dans n'importe quel navigateur web.
